# Scottish Government colours and palettes

- `sg_colour_values` is a vector containing colour names and their
  corresponding hex codes.

- `sg_colour_palettes` is a list grouping colours into palettes.

## Usage

``` r
sg_colour_values

sg_colour_palettes
```

## Format

An object of class `character` of length 11.

An object of class `list` of length 4.

## Source

[Scottish Government Design
System](https://designsystem.gov.scot/guidance/charts/data-visualisation-colour-palettes)
